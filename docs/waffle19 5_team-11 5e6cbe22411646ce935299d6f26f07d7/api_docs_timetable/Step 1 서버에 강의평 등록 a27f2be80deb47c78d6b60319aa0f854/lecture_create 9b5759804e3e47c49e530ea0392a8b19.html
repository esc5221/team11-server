lecture_create

Applecture
MethodPOST
URL/lecture/
  • 입력받은 수업의 정보를 이용해 새로운 Lecture Object을 생성한다.
    • Lecture Object는 강의에 대한 상세 정보를 저장한다.
    • SubjectProfessor Object는 과목과 교수에 대한 정보를 저장한다.
      • 각 Lecture Object는 하나의 SubjectProfessor Object를 참조한다.
        • 정보가 일치하는 SubjectProfessor Object가 있다면 해당 Object를 참조한다.
        • 그렇지 않다면, 새로운 SubjectProfessor Object가 생성된다.
  • 관리자만 시간표를 생성할 수 있다.
    • 로그인하지 않아도 시간표를 생성할 수 있다.

Query Params

None

  • subject_name(교과목명) & professor(주담당교수)
    • 위의 두 field를 이용해 SubjectProfessor Object를 불러온다.
  • subject_code(교과목번호)
  • year(연도)
  • season(학기) → {1: 1학기, 2: 2학기, 3: 여름학기, 4: 겨울학기}
  • college(대학) & department(학과)
  • grade(학년)
    • 학년 정보가 비어 있거나, 0이 적혀 있다면 0을 저장한다.
  • level(이수과정) → {1: "학사", 2: "학석사통합", 3: "석사", 4: "석박사통합", 5: "박사"}
  • credit(학점)
  • category(교과구분) → {1: '교양', 2: '전선', 3: '전필', 4: '교직', 5: '대학원', 6: '논문', 7: '일선', 8: '공통'}
  • number(강좌번호)
  • detail(비고)
  • language(언어)
  • location(장소) & time(시간)
